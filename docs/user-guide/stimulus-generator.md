
## Spikeling Stimulus Generator

<br>

<p style='text-align: justify;'>
Spikeling GUI also comes with a "Stimulus Generator" page where users can generate their own stimuli. In the current
version seven distinct stimulus pattern are available with various parameters to interact with.
All stimuli can then be saved and further loaded from the Spikeling main window.
