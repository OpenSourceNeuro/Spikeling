# Spikeling v2.2

A hardware implementation of spiking neurons for neuroscience teaching and outreach. Spikeling mimics the electrical properties of spiking neurons using the Izhikevich model on an ESP32, with a Python GUI for visualization, control, and data recording.

These docs are versioned. Use the version switcher to view other releases.
